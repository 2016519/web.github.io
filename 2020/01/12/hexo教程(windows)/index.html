hexo教程

Publicado el 2020-01-12 En hexo

安装git(查看之前的git教程)

安装node.js

node.js 官网      https://nodejs.org/en/ 下载完成之后解压即可

安装hexo

1.更换npm源
npm config set registry https://registry.npm.taobao.org npm info underscore
2.安装hexo
npm install -g hexo-cli
这里安装的是hexo最新版本，如果想安装以前的的版本请运行命令$ npm install -g hexo
3.查看版本的信息
hexo -v

创建一个hexo工程(在你想要创建的地方打开 git bash)

hexo init myblog(工程的名字)
*可以不操作这步（更新npm/并更换为国内的npm镜像命令：npm install npm -g/npm install -g cnpm --registry=https://registry.npm.taobao.org）

进入 myblog(创建的工程名字)文件夹

cd myblog

打开hexo

hexo g *生成静态文章
hexo server *开启服务

创建以用于提交

git config --global user.name "yourname"
git config --global user.email "youremail"

连接公钥

ssh-keygen -t rsa -C "你的邮箱"

查看是否连接成功

ssh -T git@github.com
出现这句就表明连接成功（You've successfully authenticated, but GitHub does not provide shell access.）

将hexo和GitHub关联起来，修改配置文件_config.yml

deploy:
type: （中间有个空格，以下相同）git
repo: 你的github地址（https://github.com/2016519/web.github.io.git）
branch: master

安装deploy-git部署命令

npm install hexo-deployer-git --save

清除之前生成的东西

hexo clean

生成静态文章

hexo g

部署文章

hexo d

github中要新建一个仓库用户名+github.io，访问用户名.github.io即可

设置hexo主题

1、选择hexo主题，到github上下载主题放到themes文件夹下   
2、修改_config.yml里的theme为该文件夹名称

hexo发布到仓库 常用的命令

-->hexo clean
-->hexo g
-->hexo d

注意！

如果更改主题后进入标签网址提示 /%20/说明在模板的配置文件中的 menu下，目录有多加的空格删除即可。

收起内容
